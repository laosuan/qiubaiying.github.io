## 快捷键

```
// Right click and choose Reveal in Project Navigator
⇧ + ⌘ + J

// History (back/forward) 
Cmd + Ctrl + ←

// Moving Lines
(⌥ + ⌘ + [ and ⌥ + ⌘ + ])

// Jump to Definition 
(⌃ + ⌘ + J or ⌃ + ⌘ + Click)

// Find Call Hierarchy
(⇧ + ⌃ + ⌘ + H)

// search all files
1 select text
2 Cmd + shift + f
```
